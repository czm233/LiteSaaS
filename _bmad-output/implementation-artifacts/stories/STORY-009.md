- **状态**: 待开始

- **预计工时**: 2小时

## 需求描述
实现字典的增删改查功能，用于存储通用的键值对配置。

### 后端
- GET `/api/v1/dicts` - 字典列表（分页）
- GET `/api/v1/dicts/types/:type` - 按类型获取字典
- POST `/api/v1/dicts` - 创建字典
- PUT `/api/v1/dicts/:id` - 更新字典
- DELETE `/api/v1/dicts/:id` - 删除字典

### 前端
- 字典管理页面：`/dicts`
- 字典类型筛选
- 字典列表（表格）
- 新建/编辑弹窗

- [ ] 可以创建新字典
- [ ] 可以编辑字典
