# STORY-010: 前端布局

## 基本信息
- **Owner**: wxp
- **状态**: 待开始
- **优先级**: P0
- **预计工时**: 3小时
- **依赖**: STORY-003

## 需求描述
实现前端基础布局，包括侧边栏、顶部导航、内容区域。

### 布局结构
```
┌─────────────────────────────────────────┐
│                  Header                 │
├────────────┬────────────────────────────┤
│            │                            │
│   Sidebar  │         Content            │
│            │                            │
│            │                            │
└────────────┴────────────────────────────┘
```

### 功能要求

#### Header
- Logo / 系统名称
- 当前租户名称
- 用户头像 + 下拉菜单
  - 个人中心
  - 修改密码
  - 退出登录

#### Sidebar
- 菜单树（根据权限动态显示）
- 支持折叠/展开
- 支持高亮当前菜单

#### Content
- 面包屑导航
- 页面内容区域

## 技术要求
- 使用 Ant Design Layout 组件
- 响应式设计
- 菜单数据从 API 获取

## 验收标准
- [ ] 布局结构正确
- [ ] Header 显示正确信息
- [ ] Sidebar 显示有权限的菜单
- [ ] 菜单可以折叠/展开
- [ ] 当前菜单高亮
- [ ] 面包屑导航正确
- [ ] 可以退出登录
