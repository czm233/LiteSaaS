## 🚨 越权处理

### 如果协作者尝试执行超出权限的命令

Claude 应该：

1. **拒绝执行**
2. **说明原因**："这个命令需要主导者权限"
3. **提供替代方案**：
   - 如果有想法，在群里提出
   - 可以查看相关文档
   - 等待主导者分配

### 示例对话

```
协作者: /bmad-create-prd

Claude: ⚠️ 这个命令需要主导者权限。

根据权限矩阵，PRD 创建只能由主导者执行。
原因：PRD 决定产品方向，需要统一决策。

你可以：
1. 在群里提出你的想法和建议
2. 查看 _bmad-output/planning-artifacts/prd/ 了解当前 PRD
3. 等待主导者分配任务

如果有紧急需求，请联系主导者 @czm。
```

## 🔄 权限变更流程

如果需要调整权限：

1. 在群里讨论
2. 更新此文档
3. 更新对应的 CLAUDE.md 模板
4. 通知所有成员
